phantomjs:install

Full name:

com.github.klieber:phantomjs-maven-plugin:0.1.0-SNAPSHOT:install

Description:

Maven plugin for downloading and installing phantomjs binaries.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: process-test-sources.

Required Parameters

Name Type Since Description
baseUrl String - The base url the phantomjs binary can be downloaded from.
Default value is: https://phantomjs.googlecode.com/files/.
User property is: phantomjs.baseUrl.
outputDirectory File - The directory the phantomjs binary should be installed.
Default value is: ${project.build.directory}/phantomjs-maven-plugin.
User property is: phantomjs.outputDir.
propertyName String - (no description)
Default value is: phantomjs.binary.
User property is: phantomjs.propertyName.
version String - The version of phantomjs to install.
User property is: phantomjs.version.
